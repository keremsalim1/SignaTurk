ML API seçenekleri Hugging Face Inference API üzerinden çalışır (RAM/CPU kullanmaz, anlık). Local ML seçenekleri ilk seferde model dosyasını indirir (HuggingFace cache); CPU üzerinde dynamic int8 quantization ile hızlandırılır.
